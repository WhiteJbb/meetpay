# 페르소나: 모임 더치페이 정산 웹

기획서([PlanningDoc.md](./PlanningDoc.md))의 결정 사항을 바탕으로 정의한
대표 사용자 페르소나.

## 페르소나 1: 모임을 만드는 사람 (정산 주최자)

- **이름(가상)**: 김지현, 32세, 회사원
- **배경**: 매달 친구 모임(4~6명)에서 총무 역할을 자주 맡는다. 밥값,
  카페, 노래방 등 지출이 여러 건 발생하는 모임을 주도한다.
- **목표**: 모임이 끝난 후 누가 얼마씩 냈는지, 최종적으로 누가 누구에게
  얼마를 보내야 하는지 빠르고 정확하게 계산해서 대화방에 공유하고
  싶다.
- **불편함(기존)**: 지출이 3~4건 이상 겹치면 손계산이 헷갈리고, 일부
  인원만 참여한 항목(예: 노래방에 안 간 사람)을 반영하면 더 복잡해진다.
- **서비스 사용 동기**: 로그인 없이 빠르게 모임을 만들고, 지출을
  입력하면 자동으로 정산 결과와 공유 링크가 나오길 원한다.
- **특징적 요구사항 연결**: 부분 참여 지정(질문1), 익명 접근(질문2),
  공유 링크(질문6)에 대한 니즈가 크다.

## 페르소나 2: 해외여행 모임 참여자

- **이름(가상)**: 박민수, 28세, 대학원생
- **배경**: 친구 3명과 해외여행을 다녀왔고, 현지 통화와 원화 결제가
  섞여 있다. 여행 중 결제한 사람이 매번 달라 정산이 번거롭다.
- **목표**: 여러 통화로 결제된 지출을 하나의 기준 통화로 환산해서
  최종 정산 금액을 명확히 알고 싶다.
- **불편함(기존)**: 엑셀로 환율을 수동 계산하다 보니 오차가 생기고,
  누가 얼마를 냈는지 기억이 안 나 다투는 경우도 있었다.
- **서비스 사용 동기**: 지출마다 통화를 선택해 입력하면 자동으로
  환산되어 정산되는 기능이 필요하다.
- **특징적 요구사항 연결**: 다국통화 지원(질문7), 영수증 첨부로 근거
  남기기(질문8)에 대한 니즈가 크다.

## 페르소나 3: 정산 결과만 확인하는 참여자

- **이름(가상)**: 이수아, 26세, 직장인
- **배경**: 모임 총무가 아니며, 모임에 참여만 하고 정산 링크를
  받아서 확인하는 입장이다.
- **목표**: 복잡한 가입 절차 없이 링크 클릭 한 번으로 내가 얼마를
  내야/받아야 하는지 바로 확인하고 싶다.
- **불편함(기존)**: 회원가입을 요구하는 정산 앱은 번거로워서 잘 쓰지
  않게 된다.
- **서비스 사용 동기**: 익명 접근, 모바일에서도 보기 편한 화면.
- **특징적 요구사항 연결**: 익명 접근(질문2), 반응형 지원(질문5),
  공유 링크(질문6)에 대한 니즈가 크다.
